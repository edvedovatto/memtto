Memtto
Memtto is a minimal, search-first personal knowledge web app.
A fast and beautiful dark-mode personal archive to store and search ideas, notes, snippets and experiences.

Tech Stack
Next.js 14
TypeScript
TailwindCSS
shadcn/ui
Supabase

Getting Started
Install dependencies
npm install
Run development server
npm run dev

Environment Variables
Create a .env.local file:
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
